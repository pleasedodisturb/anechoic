## The 5-Level Remote Maturity Model

### Level 1: No Deliberate Action
- Employees occasionally work from home
- No remote-specific policies
- Office-centric culture
- Remote workers disadvantaged

### Level 2: Recreating the Office
- Video calls replicate meetings
- Synchronous communication dominant
- "Virtual office" software used
- Still office-mindset, just online
- Two-tier system (remote vs. in-office)

### Level 3: Adapting to Async
- Async communication emerging
- Documentation becoming important
- Some processes remote-friendly
- Still some sync dependencies
- Training for remote skills

### Level 4: Async by Default
- Written communication primary
- Meetings are exception
- Documentation comprehensive
- Time zones respected
- Most work async

### Level 5: Nirvana (Automattic Level)
- **Autonomous, asynchronous communication**
- Location truly irrelevant
- Trust pervasive
- Documentation complete
- Anyone can work from anywhere anytime

### Self-Assessment Questions
- Do you require same-time communication?
- Can new hires onboard without live meetings?
- Is documentation the source of truth?
- Do remote employees have equal opportunity?
- Can work continue across all timezones?

## P2: The Communication Platform

### What is P2
- WordPress-based internal blogs
- Each team has a P2
- Async discussions in blog post format
- Comments for conversation
- Searchable, permanent record

### P2 vs. Slack
| P2 | Slack |
|----|-------|
| Long-form | Short messages |
| Permanent | Ephemeral |
| Searchable | Limited search |
| Thoughtful | Reactive |
| Async | Real-time pressure |

### Using P2
- Posts for announcements, decisions
- Comments for discussion
- Tags for organization
- Subscribe to relevant P2s
- Daily digest optional

## Communication Norms

### Writing Culture
- **Text is the primary medium**
- Long-form posts over chat
- Thoughtful, edited communication
- Assume async readers
- Context always included

### Response Expectations
- No instant response required
- Check P2 daily minimum
- Urgent matters have different channels
- Respect timezones
- Over-communicate deadlines

### When to Sync
- Complex collaborative work
- Sensitive conversations
- Relationship building
- Urgent incidents
- But always async follow-up

## Work Practices

### Grand Meetups
- **Annual company-wide gathering**
- Only time whole company is together
- Week of planning, bonding, alignment
- Expensive but valuable investment
- Creates human connections

### Team Meetups
- Teams meet in-person 1-2x per year
- Location rotates globally
- Mix of work and social
- Builds trust for async work
- Not required, but encouraged

### Creed
Automattic employees commit to:
- Writing over meeting
- Transparency over secrecy
- Iteration over perfection
- Distributed over centralized
- Asynchronous over synchronous

## Hiring & Onboarding

### Trial Period
- **2-4 week paid trial** before full hire
- Work on real projects
- Assess cultural fit both ways
- Communication skills critical
- Can end early if not fit

### What They Look For
- Excellent written communication
- Self-motivation and direction
- Async work experience
- Technical skills for role
- Cultural alignment with creed

### Onboarding
- Comprehensive documentation
- Buddy/mentor assigned
- P2 history for context
- Gradual responsibility increase
- Check-ins initially more frequent

## Key Takeaways for TPMs
