## Communication Practices

### Friday Updates
- **Weekly written update from every employee**
- Posted every Friday
- Visible to whole company
- Includes: accomplishments, plans, blockers
- Replaces status meetings entirely

### Format for Friday Update
```
## What I shipped this week
- [Accomplishment 1]
- [Accomplishment 2]

## What I'm working on next week
- [Priority 1]
- [Priority 2]

## Blockers / Help needed
- [Any blockers or asks]

## Mood
- 😊 / 😐 / 😟 (optional human element)
```

### Async Norms
- **Default to async** - Sync requires justification
- **Write to be understood by anyone** - Low context
- **Give time to respond** - 24-48h for non-urgent
- **Over-communicate** - Repeat important info
- **Thread conversations** - Keep topics organized

### When to Meet (Rare)
- Sensitive topics (performance, personal)
- Urgent incidents
- Complex brainstorming (after async prep)
- Team bonding (scheduled, optional)
- Onboarding first week

## Management Practices

### Leading Remote Teams
- **Trust by default** - No surveillance
- **Measure outcomes** - Not activity or hours
- **Over-communicate context** - Why, not just what
- **1:1s are sacred** - Never cancel
- **Document decisions** - Create institutional memory

### 1:1 Structure
- Weekly or bi-weekly
- 30 minutes minimum
- Employee sets agenda
- Mix of tactical and career
- Async pre-work expected

### Performance Management
- **Quarterly reviews** with written self-assessment
- **360 feedback** from peers
- **Clear expectations** documented upfront
- **No surprises** - ongoing feedback culture
- **Growth focus** - Development over evaluation

## Tool Purpose Clarity

### The Rule
> **Each tool has ONE primary purpose. Know what it is.**

### Zapier's Stack
| Tool | Primary Purpose |
|------|-----------------|
| Slack | Quick questions, social |
| Async (internal) | Long-form updates |
| Notion | Documentation |
| Zoom | Video when needed |
| Email | External communication |
| Zapier | Automation (of course) |

### Tool Guidelines
- Don't duplicate information across tools
- Know where to find what
- New tools require justification
- Sunset unused tools
- Train on tools properly

## Time Zone Collaboration

### Strategies
- **Overlap hours** - Find 2-3h of common time
- **Rotate meeting times** - Share the burden
- **Record everything** - Async participation
- **Clear handoffs** - End-of-day summaries
- **Respect offline time** - No after-hours expectations

### Documentation for Timezones
- Write as if reader is asleep
- Include all context in message
- Set clear deadlines with timezone
- Use async tools for non-urgent
- Schedule sends for recipient's morning

## Hiring Remote

### What to Look For
- **Self-motivation** - Works without supervision
- **Communication skills** - Especially written
- **Time management** - Structures own work
- **Tech comfort** - Can troubleshoot basics
- **Cultural fit** - Values autonomy

### Interview Process
- Mostly async (written exercises)
- One live interview for culture fit
- Work sample or trial project
- Reference checks emphasize remote experience
- Clear expectations set upfront

### Onboarding
- **First week structured** - Daily sync checkins
- **Buddy assigned** - Go-to person
- **Tool training** - Don't assume knowledge
- **Quick wins** - Ship something early
- **30/60/90 plan** - Clear milestones

## Employee Productivity

### Focus Time
- **Block calendar** for deep work
- **Communicate availability** - Slack status
- **Batch meetings** - Cluster on specific days
- **Say no** to non-essential meetings
- **Async first** - Default response

### Work-Life Boundaries
- **Set working hours** - And stick to them
- **Dedicated workspace** - Separate from living space
- **Shutdown ritual** - Clear end to day
- **Take vacation** - Actually disconnect
- **Model boundaries** - Leaders go first

### Combating Isolation
- **Optional social calls** - Donut/coffee chats
- **Team retreats** - Annual in-person
- **Interest channels** - Non-work Slack
- **Celebrate publicly** - Recognition in updates
- **Check in on wellbeing** - Part of 1:1s

## Workspace Setup

### Home Office Essentials
- Dedicated desk/room
- Reliable internet (backup recommended)
- Quality webcam and microphone
- Ergonomic chair
- Good lighting for video calls

### Zapier Provides
- Home office stipend
- Monthly internet allowance
- Hardware budget
- Coworking space allowance (optional)
